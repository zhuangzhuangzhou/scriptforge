# 改编禁忌与质量标准

## 致命错误（导致失败）

### ❌ 错误1：过度忠于原著
```
问题：试图保留小说所有内容，导致节奏拖沓
结果：观众流失，付费率低

正确做法：
✓ 大胆删减，只保留核心冲突和爽点
✓ 不要有"舍不得删"的心态
```

### ❌ 错误2：节奏过慢
```
问题：保留了小说的铺垫和过渡
结果：观众3秒就滑走了

正确做法：
✓ 3秒进冲突，瞬间抓眼球
✓ 删除所有铺垫和过渡
✓ 爽点前置，先爽再解释
```

### ❌ 错误3：心理描写过多
```
问题：保留了小说的心理独白和内心戏
结果：画面无法展现，观众看不懂

正确做法：
✓ 心理→动作（用动作展现情绪）
✓ 心理→对话（用对话说出心声）
✓ 心理→删除（不重要的直接删）
```

### ❌ 错误4：支线过多
```
问题：试图保留所有支线剧情
结果：主线模糊，观众不知道在看什么

正确做法：
✓ 只保留主线（80%）
✓ 重要支线精简保留（15%）
✓ 其他支线全部删除（5%）
```

### ❌ 错误5：人物过多
```
问题：保留了小说中的所有角色
结果：观众记不住，人物混乱

正确做法：
✓ 主角+主要配角（5-8人）
✓ 炮灰工具人合并同类型
✓ 无关角色完全删除
```

## 改编核心原则（必须遵守）

### ✅ 原则1：情绪钩子优先
```
选择标准：
- 有爽点的内容 > 完整的故事
- 能让观众"卧槽"的情节 > 逻辑严密的情节
- 情绪刺激强度 > 文学性

实施方法：
- 从小说中识别所有高强度情绪钩子
- 优先保留10-8分的钩子
- 围绕钩子重构故事
```

### ✅ 原则2：视觉化优先
```
转换标准：
- 能拍成画面的 > 抽象的描写
- 动作场景 > 心理描写
- 对话冲突 > 内心独白

实施方法：
- 所有内容必须可视化
- 用※△【】标注场景/动作/特效
- 删除无法转化为画面的内容
```

### ✅ 原则3：快节奏强制
```
节奏要求：
- 3秒进冲突
- 每30秒必有推进
- 每集必有悬念【卡黑】
- 全程无尿点

实施方法：
- 删除所有铺垫和过渡
- 场景快速切换
- 用蒙太奇压缩时间
```

### ✅ 原则4：尊重原作结构
```
设计逻辑：
- 保持小说原有故事结构
- 不强行重构三幕式
- 集数根据内容自然生成
- 单集确保起承转钩结构

实施方法：
- 按小说顺序拆解剧情
- 拆解时直接标注分集
- 不提前设定目标集数
```

### ✅ 原则5：算法友好
```
优化目标：
- 完播率：每集结尾强悬念
- 互动率：设置争议点/吐槽点
- 传播率：金句台词可截图

实施方法：
- 每集必须【卡黑】
- 人物性格/台词要有争议性
- 台词简短、霸气、可传播
```

## 质量检查清单

### 剧情拆解质检

| 检查项 | 标准 | 扣分 |
|-------|------|------|
| 剧情点格式 | 符合【剧情n】格式 | 格式错误 -5分/个 |
| 情绪钩子标注 | 每个剧情点有情绪类型 | 缺失 -3分/个 |
| 分集标注 | 每个剧情点有集数 | 缺失 -3分/个 |
| 冲突强度 | 核心冲突占比≥60% | 不足 -10分 |
| 情绪密度 | 每集至少1个高强度钩子 | 不足 -5分/集 |
| 节奏控制 | 无连续2集低强度 | 违反 -10分 |

### 剧本质检

| 检查项 | 标准 | 扣分 |
|-------|------|------|
| 字数范围 | 500-800字 | 超出/不足 -10分 |
| 起承转钩 | 四段式完整 | 缺失 -15分/段 |
| 开场冲突 | 3秒进冲突 | 铺垫过长 -10分 |
| 悬念结尾 | 必须【卡黑】 | 缺失 -20分 |
| 视觉化 | 无大段心理描写 | 违反 -5分/处 |
| 对话质量 | 简短有力 | 冗长 -3分/处 |

## 评分标准

### 总分计算
```
总分 = 100 - 扣分项总和

评级：
- 90-100分：优秀（PASS）
- 80-89分：良好（PASS）
- 70-79分：合格（PASS_WITH_WARNINGS）
- 60-69分：需改进（PASS_WITH_WARNINGS）
- 60分以下：不合格（FAIL）
```

### 质检维度权重

| 维度 | 权重 | 说明 |
|------|------|------|
| 冲突强度 | 25% | 核心冲突是否足够 |
| 情绪钩子 | 25% | 情绪密度和类型准确性 |
| 节奏控制 | 20% | 是否符合快节奏要求 |
| 格式规范 | 15% | 输出格式是否正确 |
| 视觉化 | 15% | 是否可转化为画面 |

## 常见问题修正指南

### 问题1：情绪钩子类型标注不准确
```
修正方法：
1. 重新阅读剧情点描述
2. 对照情绪钩子类型表
3. 选择最匹配的类型
4. 确保类型与内容一致
```

### 问题2：分集不合理
```
修正方法：
1. 检查每集剧情点数量（1-3个）
2. 检查每集是否有高潮
3. 高强度钩子应单独成集
4. 过渡内容应合并
```

### 问题3：冲突强度不足
```
修正方法：
1. 识别被遗漏的核心冲突
2. 删除低强度过渡内容
3. 强化现有冲突的描述
4. 确保核心冲突占比≥60%
```

### 问题4：节奏拖沓
```
修正方法：
1. 删除连续的低强度剧情点
2. 合并相似的过渡内容
3. 将铺垫内容压缩或删除
4. 确保每集有明确高潮
```

### 问题5：格式错误
```
修正方法：
1. 检查【剧情n】编号连续性
2. 确保每个剧情点包含所有要素
3. 统一情绪钩子类型命名
4. 确保分集标注格式正确
```

## 降级策略

当质检分数低于60分时，启用降级策略：

### 降级方案1：简化拆解
```
- 只保留核心冲突（⭐⭐⭐）
- 删除所有次级冲突
- 每6章只输出3-5个剧情点
- 确保每个剧情点都是高强度钩子
```

### 降级方案2：模板化输出
```
- 使用固定的剧情点模板
- 减少自由发挥
- 严格按照格式要求输出
- 优先保证格式正确
```

### 降级方案3：人工介入
```
- 标记为需要人工审核
- 输出当前最佳结果
- 附带质检报告
- 等待人工修正
```
