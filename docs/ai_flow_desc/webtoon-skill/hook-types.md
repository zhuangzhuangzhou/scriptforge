# 情绪钩子类型定义

## 概述

情绪钩子是网文改编漫剧的核心元素，是驱动观众持续观看的情绪动力。准确识别和标注情绪钩子类型是剧情拆解的关键技能。

## 钩子类型列表（7大类30+种）

### 【爽感类】
让观众感到痛快、解气、满足的情绪钩子。

| 钩子类型 | 定义 | 示例 |
|---------|------|------|
| 打脸蓄力 | 主角被轻视/嘲讽，为后续反击做铺垫 | 赘婿被岳母嘲讽没本事 |
| 打脸爽点 | 主角反击成功，让对方后悔 | 赘婿展示真实身份，岳母震惊 |
| 碾压爽点 | 主角以绝对优势压制对手 | 筑基期秒杀金丹期 |
| 绝境反杀 | 看似必死，反而逆转 | 被围杀时突破反杀 |
| 装X成功 | 低调展示实力，震惊众人 | 隐藏修为被识破 |
| 以弱胜强 | 弱势方战胜强势方 | 练气期击败元婴期 |

### 【震撼类】
让观众感到震惊、意外的情绪钩子。

| 钩子类型 | 定义 | 示例 |
|---------|------|------|
| 身份曝光 | 隐藏身份被揭露 | 穷小子是首富继承人 |
| 真相揭露 | 隐藏的真相浮出水面 | 凶手竟是身边人 |
| 反转冲击 | 剧情突然反转 | 盟友变敌人 |
| 背叛揭露 | 发现被信任的人背叛 | 好友暗中陷害 |
| 阴谋曝光 | 隐藏的阴谋被发现 | 家族灭门真相 |

### 【虐心类】
让观众感到心疼、难受的情绪钩子。

| 钩子类型 | 定义 | 示例 |
|---------|------|------|
| 虐心痛点 | 主角遭遇不幸 | 被挚爱误解 |
| 生离死别 | 与重要人物分离 | 师父为救主角牺牲 |
| 误会虐心 | 因误会而受伤 | 被当成凶手 |
| 牺牲奉献 | 为他人牺牲自己 | 以命换命救人 |
| 无奈抉择 | 两难的痛苦选择 | 救爱人还是救天下 |

### 【悬念类】
让观众产生好奇、想知道后续的情绪钩子。

| 钩子类型 | 定义 | 示例 |
|---------|------|------|
| 悬念设置 | 抛出疑问待解答 | 神秘人留下一句话 |
| 伏笔埋设 | 埋下后续线索 | 看似无关的物品 |
| 危机预告 | 预示危险将至 | 预言三天后灭门 |
| 神秘出现 | 不明身份人物登场 | 黑衣人现身相助 |

### 【成长类】
让观众感到主角在变强的情绪钩子。

| 钩子类型 | 定义 | 示例 |
|---------|------|------|
| 金手指觉醒 | 获得特殊能力 | 系统激活 |
| 实力突破 | 境界/能力提升 | 突破瓶颈 |
| 获得宝物 | 得到珍贵物品 | 获得神兵利器 |
| 人物结交 | 结识重要人物 | 拜大能为师 |
| 拜师学艺 | 获得传承指点 | 得到秘籍 |

### 【情感类】
让观众感到甜蜜、心动的情绪钩子。

| 钩子类型 | 定义 | 示例 |
|---------|------|------|
| 心动瞬间 | 产生好感的时刻 | 四目相对心跳加速 |
| 表白告白 | 直接表达爱意 | 当众表白 |
| 误会和解 | 解除误会重归于好 | 解释清楚真相 |
| 守护承诺 | 承诺保护对方 | "有我在，没人能伤你" |
| 甜蜜互动 | 温馨甜蜜的相处 | 牵手、拥抱 |

### 【冲突类】
让观众感到紧张、刺激的情绪钩子。

| 钩子类型 | 定义 | 示例 |
|---------|------|------|
| 正面对决 | 双方正面交锋 | 擂台比武 |
| 危机降临 | 危险突然到来 | 刺客杀到 |
| 被陷害 | 遭人设计陷害 | 被下毒、背锅 |
| 势力对抗 | 团体间的冲突 | 两派大战 |
| 生死危机 | 生命受到威胁 | 被追杀至绝境 |

## 钩子强度评分标准

| 分数 | 标准 | 观众反应 |
|-----|------|---------|
| 10分 | 颠覆认知、极度震撼 | "卧槽！" |
| 8-9分 | 强烈情绪冲击 | "爽！" / "虐！" / "急！" |
| 6-7分 | 有情绪波动 | "还行" |
| 5分以下 | 平淡 | 无感 |

## 使用说明

1. **唯一性原则**：每个剧情点只能标注一种主要钩子类型
2. **准确性原则**：选择最贴近剧情内容的钩子类型
3. **强度匹配**：根据观众可能的情绪反应给出合理评分
4. **避免重复**：相邻剧情点尽量避免相同钩子类型
