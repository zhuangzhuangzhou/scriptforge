# 钩子边界规则

## 概述

本节定义易混淆钩子类型的区分规则，帮助准确识别和标注钩子类型。

## 易混淆钩子区分

### 1. 打脸蓄力 vs 打脸爽点

**区分关键**：看主角的处境

- **打脸蓄力**：主角"被打脸"，处于被轻视/嘲讽的状态
  - 例："你一个废物也配？" → 蓄力
  - 例："就凭你也想挑战我？" → 蓄力

- **打脸爽点**：主角"打别人脸"，成功反击
  - 例：主角展示实力后，对方震惊道歉 → 爽点
  - 例：主角完成看似不可能的任务 → 爽点

**记忆口诀**：先被打是蓄力，反击成功是爽点。

---

### 2. 碾压爽点 vs 打脸爽点

**区分关键**：看是否有"之前被看不起"的铺垫

- **打脸爽点**：有之前的轻视/嘲讽铺垫，现在反击
  - 例：之前被嘲笑穷，现在展示财富 → 打脸爽点

- **碾压爽点**：无铺垫，直接以实力压制
  - 例：主角一出场就秒杀全场 → 碾压爽点

**记忆口诀**：有前因是打脸，无前因是碾压。

---

### 3. 真相揭露 vs 身份曝光

**区分关键**：看揭露的内容是"事"还是"人"

- **真相揭露**：揭露的是"事"，隐藏的真相
  - 例：凶手是谁、宝藏位置、阴谋内容 → 真相揭露

- **身份曝光**：揭露的是"人"，隐藏的身份
  - 例：真实姓名、隐藏地位、伪装身份 → 身份曝光

**记忆口诀**：揭事是真相，揭人是身份。

---

### 4. 悬念设置 vs 真相揭露

**区分关键**：看观众的反应

- **悬念设置**：观众"更好奇"，想知道答案
  - 例：神秘人出现，留下一句预言 → 悬念
  - 观众反应："他是谁？""会发生什么？"

- **真相揭露**：观众"恍然大悟"，明白了之前的疑问
  - 例：原来凶手是TA！→ 真相揭露
  - 观众反应："原来如此！"

**记忆口诀**：好奇是悬念，顿悟是真相。

---

### 5. 人物结交 vs 打脸蓄力

**区分关键**：看对方的态度

- **人物结交**：对方态度"友好"，有意结交
  - 例：前辈欣赏主角，主动搭话 → 结交
  - 例：强者对主角表示善意 → 结交

- **打脸蓄力**：对方态度"敌意"，轻视嘲讽
  - 例：前辈看不起主角，出言嘲讽 → 蓄力
  - 例：强者对主角表示不屑 → 蓄力

**记忆口诀**：友好是结交，敌意是蓄力。

---

### 6. 虐心痛点 vs 误会虐心

**区分关键**：看痛苦的来源

- **虐心痛点**：外部造成的伤害
  - 例：被背叛、被伤害、失去重要的人

- **误会虐心**：因误会而产生的痛苦
  - 例：被误解、冤枉、不被信任

**记忆口诀**：外伤是痛点，误解是误会。

---

### 7. 金手指觉醒 vs 实力突破

**区分关键**：看能力的来源

- **金手指觉醒**：获得外挂、系统、特殊能力
  - 例：激活系统、获得异能、觉醒血脉

- **实力突破**：常规修炼后的提升
  - 例：闭关突破、顿悟升级、功法大成

**记忆口诀**：外挂是金手指，修炼是突破。

---

### 8. 危机预告 vs 危机降临

**区分关键**：看危机是否已经发生

- **危机预告**：危险还未到来，只是预示
  - 例：预言、警告、发现异常征兆

- **危机降临**：危险已经发生，正在应对
  - 例：刺客杀到、陷阱触发、敌人包围

**记忆口诀**：预告是预警，降临是正在。

---

## 常见错误及纠正

| 错误标注 | 原因分析 | 正确标注 |
|---------|---------|---------|
| 打脸蓄力 → 打脸爽点 | 混淆了蓄力和爽点的时机 | 根据主角处境判断 |
| 真相揭露 → 身份曝光 | 没有区分"事"和"人" | 揭露人的身份用"身份曝光" |
| 悬念设置 → 伏笔埋设 | 两者都涉及信息隐藏 | 悬念是观众已知疑问，伏笔是隐藏的线索 |
| 人物结交 → 打脸蓄力 | 没有注意对方态度 | 看对方是友好还是敌意 |

## 快速判断流程

```
1. 判断是否有对抗性
   ├─ 是 → 区分蓄力/爽点/碾压
   └─ 否 → 继续下一步

2. 判断是否有信息揭露
   ├─ 是 → 区分真相/身份/悬念
   └─ 否 → 继续下一步

3. 判断是否有情绪互动
   ├─ 是 → 区分结交/虐心/情感
   └─ 否 → 继续下一步

4. 判断是否有能力提升
   ├─ 是 → 区分金手指/突破
   └─ 否 → 其他类型
```
