# 玄幻/武侠类型改编指南

## 类型特征

**核心元素**：
- 境界体系（炼气、筑基、金丹...）
- 战力数值化
- 宗门/门派体系
- 金手指/系统/传承

**核心爽点**：
- 境界突破
- 越级挑战
- 打脸羞辱者
- 身份曝光
- 宝物争夺

---

## 专属情绪钩子

| 钩子类型 | 示例 | 使用场景 |
|---------|------|---------|
| 境界突破 | 突破金丹期，气势暴涨 | 修炼后、战斗中 |
| 越级碾压 | 炼气期秒杀筑基期 | 宗门大比、挑战 |
| 身份曝光 | 原来是上古血脉 | 危机时刻、被质疑时 |
| 血脉觉醒 | 觉醒神龙血脉 | 绝境、突破瓶颈 |
| 打脸羞辱 | 当众打脸曾经羞辱自己的人 | 宗门大比、重逢 |
| 宝物争夺 | 抢到神器/丹药 | 秘境、拍卖会 |
| 传承获得 | 获得上古大能传承 | 秘境、机缘 |

## 压缩策略

### 修炼过程
```
小说：5章详细修炼描写

剧本：
【蒙太奇】
△ 叶凡盘坐修炼
【特效】灵气涌入体内
【文字】"三天后"
△ 叶凡睁眼，气势大变
【系统面板】境界：炼气三层 → 炼气九层
```

### 战斗过程
```
小说：详细的招式交换，300回合

剧本：
△ 两人身影交错
【特效】剑光与拳风碰撞
△ 叶凡一剑刺出
【慢镜】剑尖穿透护体真气
△ 对手倒飞
【系统面板】战斗结束，获得经验+1000
```

### 突破过程
```
小说：详细的突破感悟

剧本：
△ 叶凡体内真气暴涨
【特效】金光环绕全身
【系统面板】叮！突破成功！
△ 气势爆发，周围弟子被震退
众人（震惊）："他...他突破了！"
```

## 打脸节奏

**标准打脸节奏**：
- 每2-3集一次小打脸
- 每10集一次大打脸
- 第20集（付费点）最强打脸

**打脸三要素**：
1. 蓄力（被羞辱/被看不起）
2. 爆发（展示实力）
3. 反应（围观者震惊/羞辱者后悔）

**打脸示例**：
```
【蓄力】
云烟："废物，你永远不可能赢我。"

【爆发】
△ 叶凡一剑斩出
【特效】剑气撕裂虚空
△ 云烟被震飞

【反应】
众人（震惊）："这...这是什么剑法！"
云烟（不敢置信）："不可能..."
```
