# 悬疑/推理类型改编指南

## 类型特征

**核心元素**：
- 谜团与真相
- 线索与推理
- 紧张与危机
- 反转与揭秘

**核心爽点**：
- 凶手曝光
- 真相反转
- 关键线索发现
- 危机时刻
- 最后一搏

---

## 专属情绪钩子

| 钩子类型 | 示例 | 使用场景 |
|---------|------|---------|
| 凶手曝光 | 原来凶手是身边人 | 案件结尾 |
| 真相反转 | 以为的受害者其实是凶手 | 剧情高潮 |
| 线索发现 | 发现关键证据 | 调查过程 |
| 危机时刻 | 被凶手发现/追杀 | 紧张场景 |
| 时间压力 | 倒计时/限时救人 | 高潮前 |
| 身份揭露 | 卧底身份曝光 | 反转时刻 |

---

## 改编重点

### 必保留内容
```
✓ 关键线索发现场景
✓ 反转时刻
✓ 真相揭露场景
✓ 紧张追逐场景
✓ 凶手曝光场景
✓ 危机对峙场景
```

### 必删除内容
```
❌ 推理过程（过度详细的）
❌ 细节描写（无关的）
❌ 日常调查琐事
❌ 技术分析过程
❌ 法医报告细节
```

### 必强化内容
```
⚡ 悬念设置（每集结尾）
⚡ 反转冲击（出人意料）
⚡ 紧张氛围（音效/画面）
⚡ 时间压力（倒计时）
```

---

## 压缩策略

### 推理过程
```
小说：详细的推理分析

剧本：
△ 陈探看着证据
【闪回】案发现场的细节
△ 陈探眼神一亮
陈探："我知道了。"
助手："什么？"
陈探："凶手...就在我们身边。"
【卡黑】
```

### 调查细节
```
小说：详细的调查取证过程

剧本：
【蒙太奇】
△ 陈探查看监控
△ 陈探询问证人
△ 陈探发现关键物证
【文字】"三天后"
陈探："证据齐了。"
```

### 技术分析
```
小说：详细的法医/技术分析

剧本：
法医："死亡时间是昨晚11点。"
陈探："还有呢？"
法医："凶器是一把特制的刀，全市只有三家店卖。"
△ 陈探转身离开
```

## 悬念设置技巧

### 每集结尾悬念
```
类型1：新线索
△ 陈探发现一张照片
【特写】照片上的神秘人影
陈探："原来...还有第三个人。"
【卡黑】

类型2：危机
△ 陈探独自进入仓库
△ 身后传来脚步声
神秘人："你知道得太多了。"
【卡黑】

类型3：反转
△ 陈探看着DNA报告
△ 瞳孔一缩
陈探："不可能...凶手是..."
【卡黑】
```

### 反转设计
```
第一层反转：嫌疑人A不是凶手
第二层反转：嫌疑人B也不是凶手
第三层反转：真凶是意想不到的人C
```

## 紧张氛围营造

### 追逐场景
```
△ 陈探在黑暗中奔跑
【音效】急促的脚步声
△ 身后的追击者越来越近
陈探（喘气）："该死..."
△ 前方是死胡同
△ 陈探转身，面对追击者
```

### 对峙场景
```
△ 陈探与凶手面对面
凶手："你以为你赢了？"
△ 凶手掏出刀
陈探："放下武器，你跑不掉的。"
凶手（冷笑）："那就一起死吧。"
```

### 倒计时场景
```
【文字】"距离爆炸还有5分钟"
△ 陈探疯狂拆除炸弹
【特写】倒计时：4:59...4:58...
陈探："红线还是蓝线..."
【文字】"3分钟"
```
