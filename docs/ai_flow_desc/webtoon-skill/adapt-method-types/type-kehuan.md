# 科幻/末世类型改编指南

## 类型特征

**核心元素**：
- 末世危机（丧尸/怪物/灾难）
- 异能/变异
- 基地/势力争夺
- 人性考验

**核心爽点**：
- 异能觉醒
- 危机爆发
- 基地争夺
- 黑科技展示
- 团队对抗

---

## 专属情绪钩子

| 钩子类型 | 示例 | 使用场景 |
|---------|------|---------|
| 异能觉醒 | 觉醒SSS级火焰异能 | 危机时刻、绝境 |
| 危机爆发 | 丧尸潮来袭 | 开场、转折点 |
| 基地争夺 | 攻占敌方基地 | 中后期剧情 |
| 黑科技 | 展示超级武器 | 战斗场景 |
| 人性考验 | 背叛/牺牲 | 团队冲突 |
| 终极一战 | 对抗最终Boss | 高潮结尾 |

## 改编重点

### 必保留内容
```
✓ 危机爆发场景
✓ 异能觉醒场景
✓ 黑科技展示
✓ 基地争夺战
✓ 丧尸/怪物对抗
✓ 人性考验场景
✓ 团队协作战斗
```

### 必删除内容
```
❌ 世界观详细铺垫
❌ 科学原理解释
❌ 环境描写
❌ 日常求生琐事
❌ 物资收集细节
❌ 基地建设过程
```

### 必强化内容
```
⚡ 视觉特效（异能/科技）
⚡ 紧张节奏（危机持续）
⚡ 危机升级（不断加强）
⚡ 团队配合（战斗场景）
```

## 压缩策略

### 世界观建立
```
小说：详细的末世背景介绍

剧本：
【文字】"2030年，病毒爆发，人类文明崩溃"
【全景】废墟城市，丧尸游荡
△ 主角躲在废弃建筑中
【独白】这是末世第100天
```

### 科技原理
```
小说：详细解释黑科技原理

剧本：
科学家："这是最新研发的等离子武器。"
△ 按下按钮
【特效】蓝色光束射出，丧尸瞬间蒸发
主角："好用就行。"
```

### 求生过程
```
小说：详细的物资收集、基地建设

剧本：
【蒙太奇】
△ 团队搜索物资
△ 加固基地防御
△ 训练战斗技能
【文字】"一个月后"
△ 基地初具规模
```

## 异能系统设计

### 等级展示
```
【系统面板】
异能者：林风
异能：火焰（S级）
等级：Lv.5
技能：
- 火球术
- 火焰护盾
- 烈焰风暴（未解锁）
```

### 觉醒场景
```
△ 林风被丧尸包围
△ 绝望中，体内涌出一股热流
【特效】火焰从体内爆发
【系统提示】"恭喜！觉醒S级异能：火焰"
△ 火焰席卷，丧尸化为灰烬
△ 林风看着自己的双手
林风："这是...什么力量？"
```

## 危机升级设计

**危机等级**：
- Lv.1：普通丧尸/小型怪物
- Lv.2：变异丧尸/中型怪物
- Lv.3：丧尸王/大型怪物
- Lv.4：丧尸潮/怪物群
- Lv.5：终极Boss

**升级节奏**：
- 第1-5集：Lv.1-2危机
- 第6-10集：Lv.2-3危机
- 第11-15集：Lv.3-4危机
- 第16-20集：Lv.4危机 + Boss预告

## 团队战斗设计

### 配合展示
```

林风："我来开路！"
【特效】火焰横扫
张雷："掩护！"
△ 举起盾牌挡住攻击
李冰："看我的！"
【特效】冰墙升起，阻挡丧尸
△ 团队突围成功
```

```
△ 物资只够一人存活
队友A："你走吧，我来断后。"
林风："不行！我们一起走！"
队友A："别傻了...活下去，替我报仇。"
△ 队友A冲向丧尸群
林风："不——！"
```
