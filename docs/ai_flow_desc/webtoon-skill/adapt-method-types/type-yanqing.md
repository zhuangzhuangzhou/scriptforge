# 言情/古言类型改编指南

## 类型特征

**核心元素**：
- 误会与真相
- 虐恋情深
- 身份差距
- 三角关系

**核心爽点**：
- 误会解除
- 虐心痛苦
- 真心表白
- 破镜重圆
- 吃醋争风

---

## 专属情绪钩子

| 钩子类型 | 示例 | 使用场景 |
|---------|------|---------|
| 误会产生 | 看到男主和别的女人在一起 | 关系建立后 |
| 虐心痛点 | 被迫分离、生离死别 | 感情高潮 |
| 真相揭露 | 发现他一直在暗中守护 | 误会解除 |
| 甜宠时刻 | 霸道宠溺、独占欲 | 感情升温 |
| 吃醋争风 | 看到情敌出现 | 三角关系 |
| 破镜重圆 | 分手后重逢 | 后期剧情 |

## 改编重点

### 必保留内容
```
✓ 误会产生场景
✓ 虐心痛苦场景
✓ 真相大白场景
✓ 甜宠片段
✓ 吃醋争风场景
✓ 表白/求婚场景
✓ 生离死别场景
```

### 必删除内容
```
❌ 心理独白（过度的）
❌ 日常甜蜜（重复的）
❌ 环境描写
❌ 无关支线
❌ 配角感情线（次要的）
```

### 必强化内容
```
⚡ 情感冲突（对话/表情）
⚡ 误会解除（反转时刻）
⚡ 虐心时刻（泪点）
⚡ 反差对比（虐转甜）
⚡ 眼神交流（特写）
```

## 压缩策略

### 心理活动
```
小说：大段内心独白

剧本：
△ 苏晚看着远去的背影，眼眶泛红
【独白】为什么...你不肯告诉我真相
△ 一滴泪落下
```

### 日常甜蜜
```
小说：多章日常互动

剧本：
【蒙太奇】
△ 两人一起吃饭
△ 两人一起散步
△ 男主为女主披上外套
【文字】"就这样，过了三个月"
```

### 误会铺垫
```
小说：慢慢铺垫误会

剧本：
△ 苏晚推开门
△ 看到顾深和另一个女人拥抱
△ 苏晚瞳孔一缩
【特写】眼中的泪光
△ 转身跑开
```

## 虐心节奏设计

**虐心三要素**：
1. 建立感情（让观众代入）
2. 制造误会/分离（虐点）
3. 真相/重逢（释放）

**虐心节奏**：
- 第1-5集：建立感情，甜蜜互动
- 第6-10集：误会产生，开始虐心
- 第11-15集：虐心加深，痛苦分离
- 第16-20集：真相揭露，破镜重圆

**虐心程度控制**：
```
轻虐：误会、吃醋、冷战
中虐：分离、背叛（假的）、第三者
重虐：生死、牺牲、永别（假的）
```

## 甜宠技巧

**霸道宠溺**：
```
顾深："从今以后，你只能看我一个人。"
△ 将苏晚拥入怀中
顾深："谁敢欺负你，我灭了他全家。"
```

**反差萌**：
```
△ 顾深在外人面前冷若冰霜
下属："顾总，您的咖啡。"
顾深："出去。"

△ 苏晚进来
顾深（瞬间变脸）："晚晚，饿不饿？想吃什么？"
△ 下属石化
```

**独占欲**：
```
△ 有人靠近苏晚
顾深（冷声）："离她远点。"
△ 将苏晚护在身后
顾深："她是我的。"
```

## 经典台词模板

```
虐心类：
- "对不起，我不能告诉你真相。"
- "忘了我吧，这样对你好。"
- "如果重来一次，我不会再放手。"

甜宠类：
- "全世界，你最重要。"
- "我的眼里，只有你。"
- "欺负你的人，我一个都不会放过。"

误会解除类：
- "原来...你一直在保护我。"
- "为什么不早点告诉我？"
- "傻瓜，我怎么会不要你。"

表白类：
- "我喜欢你，从第一眼开始。"
- "余生，只想和你一起。"
- "嫁给我。"
```
