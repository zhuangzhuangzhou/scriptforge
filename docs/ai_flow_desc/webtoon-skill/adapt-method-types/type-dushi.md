# 都市/现代类型改编指南

## 类型特征

**核心元素**：
- 身份反差（隐藏富豪/高手）
- 豪门恩怨
- 商战对决
- 现代社会规则

**核心爽点**：
- 身份揭露
- 财富碾压
- 地位反转
- 真相大白
- 前妻/前女友后悔

---

## 专属情绪钩子

| 钩子类型 | 示例 | 使用场景 |
|---------|------|---------|
| 身份揭露 | 原来是隐藏的豪门继承人 | 被羞辱后、危机时刻 |
| 财富对比 | 一张黑卡震惊全场 | 购物、拍卖、还债 |
| 地位碾压 | 总裁亲自迎接 | 公司、宴会 |
| 真相大白 | 揭穿小三/渣男真面目 | 家庭聚会、公司 |
| 商战胜利 | 收购对方公司 | 商业谈判 |
| 前任后悔 | 前妻发现真实身份 | 偶遇、宴会 |

## 改编重点

### 必保留内容
```
✓ 身份反差场景（豪门vs屌丝）
✓ 财富对比场景（贫富差距）
✓ 地位碾压场景（身份揭露）
✓ 真相大白场景（误会解除）
✓ 商战对决场景
✓ 前任后悔场景
```

### 必删除内容
```
❌ 日常生活琐事
❌ 工作细节描写
❌ 心理独白
❌ 城市风景描写
❌ 商业谈判细节
❌ 法律程序描写
```

### 必强化内容
```
⚡ 对话冲突（语言交锋）
⚡ 身份反转（视觉冲击）
⚡ 打脸时刻（众人反应）
⚡ 财富展示（具体数字）
```

## 身份反差设计

**常见身份设定**：
- 隐藏富豪（装穷三年）
- 退役特种兵（扮猪吃虎）
- 豪门私生子（身份曝光）
- 神医/高手（隐藏实力）

**身份揭露三要素**：
1. 被羞辱/看不起
2. 关键时刻身份曝光
3. 羞辱者震惊/后悔

---

## 财富展示技巧

**具体化展示**：
```
❌ "他很有钱"

✅
服务员："先生，这块表价值3000万。"
沈浩："包起来，再来十块。"
△ 众人倒吸一口凉气
```

**对比展示**：
```
前妻："你连房租都交不起！"
△ 沈浩拿出一张黑卡
银行经理（恭敬）："沈先生，您的账户余额是...50亿。"
△ 前妻石化
```
