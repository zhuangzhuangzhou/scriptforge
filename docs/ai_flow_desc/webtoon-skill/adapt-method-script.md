# 单集剧本创作方法论

## 剧本创作核心

**任务**：将剧情点转化为500-800字的单集剧本

**输出格式**：使用标准漫剧剧本格式，包含场景、动作、对话、特效标注

---

## 剧本格式规范

### 场景标注
```
※ 场景名称（时间）

示例：
※ 宗门大殿（白天）
※ 悬崖边（深夜）
※ 王府书房（黄昏）
```

### 动作描写
```
△ 动作描述

示例：
△ 叶凡缓缓站起，眼神冰冷
△ 云烟将婚书撕成碎片，扔在叶凡脚下
△ 众人倒吸一口凉气
```

### 对话格式
```
角色名："对话内容"

示例：
云烟："你不过是个炼气三层的废物！"
叶凡："三天后，你会后悔今天说的每一个字。"
```

### 特效标注
```
【特效类型】描述

示例：
【特写】叶凡眼中闪过一丝寒光
【闪回】三年前的婚约场景
【蒙太奇】修炼过程快速切换
【文字】"三天后"
【卡黑】
```

### 内心独白
```
【独白】内容

示例：
【独白】这次，我不会再让任何人看不起我
```

## 单集结构模板

### 起承转钩四段式

```
※ 场景一（开场）

【起】开场冲突（100-150字）

△ 动作描述
角色A："冲突对话"
角色B："回应"

---

※ 场景二（发展）

【承】推进发展（150-200字）

△ 动作描述
【特效】展示
角色对话推进

---

※ 场景三（高潮）

【转】反转高潮（200-250字）

△ 核心爽点动作
【特效】震撼展示
角色A："高潮台词"
△ 众人反应

---

【钩】悬念结尾（100-150字）

△ 新危机/新角色出现
角色："悬念台词"

【卡黑】
```

## 对话写作原则

### 原则1：简短有力
```
❌ "我已经在这个宗门修炼了整整十年，从来没有人像你这样羞辱过我，今天你当着所有人的面撕毁婚约，我一定会让你后悔的。"

✅ "十年。你会后悔今天。"
```

### 原则2：冲突明确
```
❌ "我觉得你可能不太适合我们宗门的修炼方式。"

✅ "废物，滚出宗门！"
```

### 原则3：情绪外化
```
❌ 叶凡心想：我一定要变强。

✅ 叶凡："看好了，这就是你们口中的废物！"
```

### 原则4：金句可传播
```
经典台词模板：
- "三天后，你会跪着求我。"
- "从今天起，规则由我来定。"
- "你以为这就是我的全部实力？"
- "记住这一刻，因为这是你最后一次看不起我。"
```

## 视觉化转换

### 心理描写 → 动作/对话
```
小说：叶凡心中充满了愤怒，他发誓一定要让这些人付出代价。

剧本：
△ 叶凡拳头紧握，指节发白
△ 眼中闪过一丝杀意
叶凡（低声）："记住今天。"
```

### 环境描写 → 快速建立
```
小说：宗门大殿金碧辉煌，巨大的石柱上雕刻着古老的符文，阳光透过穹顶洒下，照亮了殿中央的祭坛...（200字）

剧本：
※ 宗门大殿（白天）
【全景】恢弘大殿，众人林立
```

### 战斗描写 → 关键招式
```
小说：两人你来我往，打了三百回合...（1000字）

剧本：
△ 两人身影交错
【特效】剑光闪烁
△ 叶凡一剑刺出
【慢镜】剑尖穿透对方护体真气
△ 对手倒飞出去
```

## 场景切换技巧

### 硬切
```
※ 场景A

（内容）

---

※ 场景B

（内容）
```

### 闪回
```
【闪回开始】
※ 三年前·婚约现场

△ 年轻的叶凡满怀期待

【闪回结束】

※ 现在·宗门大殿

△ 叶凡眼神冰冷
```

### 蒙太奇
```
【蒙太奇开始】

△ 叶凡修炼·第一天
△ 叶凡修炼·第三天
△ 叶凡修炼·第七天
【文字】"七天后"

【蒙太奇结束】
```

## 悬念结尾设置

### 类型1：战斗悬念
```
△ 更强的敌人出现
神秘人："有意思，让我来会会你。"
△ 叶凡瞳孔一缩
【卡黑】
```

### 类型2：身份悬念
```
△ 叶凡露出某个标记
长老（震惊）："这...这是..."
【卡黑】
```

### 类型3：危机悬念
```
△ 远处传来爆炸声
弟子（慌张）："不好！魔族入侵了！"
【卡黑】
```

### 类型4：真相悬念
```
△ 叶凡发现一封信
【特写】信上的内容
叶凡："原来...是你。"
【卡黑】
```

---

## 字数控制

**每集500-800字分配**：
- 【起】开场：100-150字
- 【承】发展：150-200字
- 【转】高潮：200-250字
- 【钩】悬念：100-150字

**字数统计方法**：
- 场景标注不计入
- 特效标注不计入
- 只统计动作描写和对话

## 剧本示例

```
※ 宗门大殿（白天）

【起】

△ 大殿中央，云烟手持婚书，面带嘲讽

云烟："叶凡，你我的婚约，到此为止。"

△ 她将婚书撕成碎片

云烟："你不过是个炼气三层的废物，配不上我云家的女儿！"

△ 众人哄笑

弟子甲："哈哈，废物也想娶云家小姐？"

---

【承】

△ 叶凡站在原地，一言不发

云烟："怎么？不服气？"

△ 她走近叶凡，居高临下

云烟："三天后的宗门大比，你要是能进前十，我就收回这句话。"

△ 众人再次哄笑

弟子乙："炼气三层进前十？做梦吧！"

---

【转】

△ 叶凡缓缓抬头，眼中闪过一丝寒光

叶凡："好。"

△ 众人一愣

叶凡："三天后，你会跪着求我收回这句话。"

△ 全场寂静

云烟（冷笑）："我倒要看看，你有什么本事。"

△ 叶凡转身离去，背影决绝

---

【钩】

※ 叶凡住处（夜晚）

△ 叶凡盘坐修炼，突然眼前出现一道光芒

【特效】系统面板浮现

【文字】"签到系统已激活"

△ 叶凡嘴角微微上扬

叶凡："有意思。"

【卡黑】
```
