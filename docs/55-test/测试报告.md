### 4. 用户认证测试

#### 4.1 认证API测试（2026-02-20 第二轮）

**问题：数据库连接不可用**

测试时发现数据库连接失败，错误信息：
```
asyncpg.exceptions._base.InterfaceError: connection is closed
ConnectionRefusedError: [Errno 61] Connection refused
```

**原因分析**：
- 后端服务正在运行（进程85514，端口8000）
- 但SSH隧道未建立，端口5433（PostgreSQL）无法访问
- .env配置显示数据库通过SSH隧道连接到远程服务器REMOVED_IP

**SSH隧道命令**：
```bash
ssh -o ServerAliveInterval=60 -L 5433:127.0.0.1:35432 -L 6380:127.0.0.1:6379 -L 9000:127.0.0.1:19000 root@REMOVED_IP
```

| 接口 | 端点 | 状态 | 备注 |
|------|------|------|------|
| 用户登录 | POST /api/v1/auth/login | ⚠️ 无法测试 | 数据库连接失败 |
| 用户注册 | POST /api/v1/auth/register | ⚠️ 无法测试 | 数据库连接失败 |
| 获取当前用户 | GET /api/v1/auth/me | ⚠️ 无法测试 | 需要认证token |
| 获取用户余额 | GET /api/v1/auth/balance | ⚠️ 无法测试 | 需要认证token |

**结论**：需要先建立SSH隧道才能继续测试
