## ⚠️ 常见问题

### Q1: 数据库迁移失败
**A:** 检查数据库连接和权限，确保用户有创建表的权限。

### Q2: API Key 加密失败
**A:** 确认 ENCRYPTION_KEY 环境变量已正确设置。

### Q3: 前端页面无法访问
**A:** 检查路由配置和权限验证，确认用户是管理员。

### Q4: API 调用返回 403
**A:** 检查用户权限，确认是管理员角色。

### Q5: 配置更改不生效
**A:** 检查是否需要重启服务，某些配置可能需要重启。
