## 📖 如何使用文档

### 新用户
1. 从 [快速开始指南](01-getting-started/quickstart.md) 开始
2. 阅读 [安装指南](01-getting-started/installation.md) 完成环境搭建
3. 查看 [配置指南](01-getting-started/configuration.md) 进行系统配置

### 开发者
1. 阅读 [架构总览](03-architecture/overview.md) 了解系统设计
2. 根据技术栈查看 [后端开发指南](04-development/backend-guide.md) 或 [前端开发指南](04-development/frontend-guide.md)
3. 参考 [API 参考](04-development/api-reference.md) 进行开发

### 运维人员
1. 查看 [部署指南](06-deployment/deployment-guide.md) 了解部署流程
2. 参考 [Docker 配置](06-deployment/docker-setup.md) 进行容器化部署
3. 遇到问题时查阅 [故障排查](06-deployment/troubleshooting.md)
